# almarjan-finance

Sistem Manajemen Keuangan Internal PAUD Al-Marjan.

Aplikasi berbasis web sederhana untuk mengelola pencatatan keuangan internal sekolah, meliputi pencatatan penerimaan SPP bulanan, biaya tahunan, pengeluaran kas operasional, administrasi siswa, dan ekspor berkas LPJ.

## Tech Stack
- PHP (Laravel Framework)
- Bootstrap 5
- AlpineJS
- SQLite (Lokal) / PostgreSQL (Produksi)
